# CalcAdhesiveForce
粘着力計測データの解析コード

## 環境
OS:windows

Pyhton 3.7

## 内容
* CSVファイルを読み込んでオフセット値との差分を表示する
* 入力するデータは16列，1行ごとのデータ
* ファイルパス指定
* 各フレームの画像を生成し保存する

## 今後の実装内容
* GUIの作成
  * 計測データの選択を簡単に
  * 粘着力のピーク値を表示
  * 複数の計測データを一度に参照
* 出力画像のフォルダを作成
* 縦軸の単位を指定する
  * キャリブレーション結果から1ポイントあたりの力（mN）を計算
  * mN/計測値をかける
